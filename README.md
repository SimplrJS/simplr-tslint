# simplr-tslint
A set of [TSLint](https://palantir.github.io/tslint/) rules used in SimplrJS projects.

# Get started
```cmd
npm install simplr-tslint --save-dev
```

# How to use?
Add this line in your `tslint.json` file: 
```json
{
    "extends": "simplr-tslint"
}
```
